    # MoCA, Shanghai, China

    # 01.09.2019

    2019年1月9日，音乐人、新锐艺术家Angelus Novus（蔡悦宁）与上海新媒体艺术团体分号C（；C）在上海的中心枢纽——人民广场，上海当代艺术馆的亭台进行了一场行为及视听表演。通过实时的摄像反馈和影像生成，在涵盖UK Dubstep、Techno和Ambient的DJ set中，以多媒体、跨感官的表演手法与当下的城市现实产生对话，并激发对于城市空间的新的感知。
